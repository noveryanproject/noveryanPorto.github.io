## ⚙️ Cara Menjalankan Secara Lokal

### 1. Clone repository

```bash
git clone https://github.com/noveryanproject/noveryanPorto.github.io.git
```

### 2. Masuk ke folder

```bash
cd noveryanPorto.github.io
```

### 3. Install dependencies

```bash
npm install
```

### 4. Jalankan Tailwind CLI

```bash
npx tailwindcss -i ./src/css/input.css -o ./public/output.css --watch
```

### 5. Buka di browser

Buka file `index.html`
